Setup

Esta app comprueba que tienes en el equipo e instala lo que falte (runtime, modelo desde Hugging Face, plugin de Resolve).

Necesitas Python 3.10+ en PATH y conexion a Internet para el modelo. GPU: NVIDIA usa CUDA; AMD sin Vulkan SDK (completo, no solo runtime) instala CPU automaticamente.
